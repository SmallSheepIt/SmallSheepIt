## 1、运行对话框相关命令

下面命令是使用 win+R 调出运行对话框，可输入在运行对话框的相关命令

#### 1.1、查看|打开系统配置

| CMD          | 作用                                                         |
| ------------ | ------------------------------------------------------------ |
| msinfo32     | 打开系统信息工具，提供计算机和系统组件的详细信息             |
| dxdiag       | 打开DirecX诊断工具，可查看【系统】【显卡】【声卡】【输入设备】 |
| slui         | 查看系统激活信息                                             |
| control      | 打开控制面板                                                 |
| compmgmt.msc | 打开计算机管理                                               |
| diskmgmt.msc | 打开磁盘管理                                                 |
| devmgmt.msc  | 打开设备管理器                                               |
| sysdm.cpl    | 打开【系统属性】，可设置环境变量                             |
| taskmgr      | 任务管理器                                                   |

#### 1.2、打开系统软件

| CMD      | 作用            |
| -------- | --------------- |
| explorer | 打开资源管理器  |
| clac     | 打开计算器      |
| cmd      | 打开cmd命令窗口 |
